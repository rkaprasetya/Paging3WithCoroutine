# Paging3WithCoroutine
an app fetching data and display the list using paging 3. 

Design pattern : MVVM + Clean Architecture

Programming Language : Kotlin

Libraries used :
- Retrofit
- Coroutine
- daggerhilt
- navigation component
- Paging3
- Coil
- MockWebServer
- Espresso
